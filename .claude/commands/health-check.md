# Project Health Check

Perform a comprehensive health check on all project components.

## Checks to perform:
1. **Code Quality**
   - Test coverage percentage
   - Linting issues
   - Code duplication
   - Cyclomatic complexity

2. **Dependencies**
   - Outdated packages
   - Security vulnerabilities
   - License compatibility

3. **Infrastructure**
   - Docker containers status
   - Database connectivity
   - Temporal workflows health
   - API endpoints responding

4. **Data Quality**
   - Scraper success rates
   - Data freshness
   - Missing required fields
   - Anomaly detection

5. **Performance**
   - API response times
   - Database query performance
   - Memory usage
   - CPU utilization

Generate a health report with recommendations for improvements.

# Scraper Testing Agent

Test and validate a specific scraper implementation.

## Tasks:
1. Run the scraper with sample data
2. Validate output format matches data model
3. Check rate limiting is working
4. Test error recovery mechanisms
5. Verify data quality and completeness
6. Generate test report with metrics

## What to test:
- Data extraction accuracy
- Handling of missing fields
- Pagination logic
- Rate limit compliance
- Proxy rotation if applicable
- Change detection
- Error recovery

Please specify which scraper to test (bayut, dubizzle, carrefour, etc.)
